Artificial ScrollUp is an awesome plugin for your wordpress website. 

== Description ==

This plugin will enable bottom to top page scroll system in your wordpress theme.

Wanna see how it works? Click here: http://rapiditsolution.ciki.me/rapid/our-free-plugin/

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Than your work is done enjoy this plugin.


== Frequently Asked Questions ==

= How do I install this plugin? =

You can install as others regular wordpress plugin. No different way. Please see on installation tab.
